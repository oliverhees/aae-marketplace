# AAE Marketplace

Claude Code Plugin Marketplace by **AI Automation Engineers**.

## Plugins

### saas-agency

Complete AI agency with **38 specialized agents** for autonomous SaaS development.

**Teams included:**
- C-Level & Management (CEO, CTO, COO, Project Manager)
- Design Team (UX Designer, UI Designer, Brainstormer, Copywriter)
- Development Team (Frontend/Backend Leads & Developers, API Architect, Performance Engineer)
- QA Team (QA Lead, Visual QA, Code Reviewer, Testing Specialist)
- DevOps Team (DevOps Engineer, Infrastructure Architect, Monitoring Specialist)
- Security Team (Security Specialist, Auth Expert, Accessibility Expert, Privacy Expert)
- Tech Specialists (Supabase, Vercel, AI Integration, Qdrant, Stripe, Email, Coolify, Redis, Next.js)
- Analytics & Content (Data Analyst, Analytics Specialist, SEO Specialist, Content Strategist)

**Tech Stack:** Next.js 16, Supabase, ShadcnUI, Vercel, Playwright

## Installation

```bash
# Add marketplace
/plugin marketplace add github:oliverhees/aae-marketplace

# Install plugin
/plugin install saas-agency@aae-marketplace
```
